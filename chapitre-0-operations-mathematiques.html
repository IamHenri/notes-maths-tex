Calcul littéral — Fiche de cours

Définitions

  • Une expression littérale contient des nombres et des lettres (inconnues ou variables).
  • Exemples : \(3x + 5\), \(a^2 - 4b\), \(2(x + 1)\).

Développement

\[ a(b + c) = ab + ac \]

\[ (a + b)^2 = a^2 + 2ab + b^2 \]

\[ (a - b)^2 = a^2 - 2ab + b^2 \]

\[ (a + b)(a - b) = a^2 - b^2 \]

\[ (kx + c)(mx + d) = kmx^2 + (kd + cm)x + cd \]

Factorisation

\[ ab + ac = a(b + c) \]

\[ a^2 - b^2 = (a + b)(a - b) \]

\[ a^2 + 2ab + b^2 = (a + b)^2 \]

\[ a^2 - 2ab + b^2 = (a - b)^2 \]

Identités remarquables

Nom Forme développée Forme factorisée
Carré d'une somme \(a^2 + 2ab + b^2\) \((a + b)^2\)
Carré d'une différence \(a^2 - 2ab + b^2\) \((a - b)^2\)
Différence de deux carrés \(a^2 - b^2\) \((a + b)(a - b)\)

Réduction

\[ 3x + 2x = 5x \]

\[ 7a - 2a = 5a \]

\[ 2x + 3y - x + y = x + 4y \]

Substitution

Si \(x = 2\), alors :

\[ 3x + 5 = 3 \times 2 + 5 = 11 \]

\[ (x + 1)^2 = (2 + 1)^2 = 9 \]

Exercices types

  1. Développer : \(2(x + 3)\)
  2. Factoriser : \(5x + 10\)
  3. Réduire : \(4a + 2b - a + 3b\)
  4. Calculer pour \(x = 4\) : \(x^2 - 2x + 1\)
