Introduction aux probabilités — Niveau Terminale

1. Expérience aléatoire

Définition :
Une expérience aléatoire est une expérience dont le résultat dépend du hasard.
Exemples : lancer un dé, tirer une carte, effectuer un test médical.

2. Univers et événements

Univers :
L'univers \( \Omega \) est l'ensemble de toutes les issues possibles d'une expérience.
Événement :
Un événement est un sous-ensemble de \( \Omega \).
Pour un lancer de dé : \[ \Omega = \{1,2,3,4,5,6\} \] Si \( A \) est l'événement "obtenir un nombre pair" : \[ A = \{2,4,6\} \]

Exercices corrigés — Univers et événements

Exercice 1 — Lancer d’un dé

On lance un dé équilibré à six faces.

1. Déterminer l’univers \( \Omega \).
2. Décrire les événements suivants :

  • \( A \) : « obtenir un nombre pair »
  • \( B \) : « obtenir un nombre supérieur ou égal à 4 »
  • \( A \cap B \)
  • \( A \cup B \)

Correction :

1. L’univers est : \[ \Omega = \{1,2,3,4,5,6\} \]

2. \[ A = \{2,4,6\} \] \[ B = \{4,5,6\} \]

Intersection : \[ A \cap B = \{4,6\} \]

Union : \[ A \cup B = \{2,4,5,6\} \]

Exercice 2 — Tirage d’une carte

On tire une carte dans un jeu de 32 cartes.

1. Déterminer l’univers \( \Omega \).
2. Décrire les événements :

  • \( C \) : « tirer un cœur »
  • \( D \) : « tirer une figure (Valet, Dame, Roi) »
  • \( C \cap D \)

Correction :

1. L’univers est constitué des 32 cartes du jeu : \[ |\Omega| = 32 \]

2. Il y a 8 cartes par couleur. \[ C = \text{ensemble des 8 cartes de cœur} \]

Il y a 3 figures par couleur, donc : \[ D = \text{ensemble des 12 figures} \]

Intersection : \[ C \cap D = \{\text{Valet, Dame, Roi de cœur}\} \]

Exercice 3 — Deux lancers de pièce

On lance deux fois une pièce équilibrée.

1. Déterminer l’univers \( \Omega \).
2. Décrire les événements :

  • \( E \) : « obtenir exactement une fois pile »
  • \( F \) : « obtenir au moins une fois pile »
  • \( \overline{F} \)

Correction :

1. L’univers est : \[ \Omega = \{(P,P),(P,F),(F,P),(F,F)\} \]

2. \[ E = \{(P,F),(F,P)\} \]

\[ F = \{(P,P),(P,F),(F,P)\} \]

Complémentaire : \[ \overline{F} = \{(F,F)\} \]

3. Probabilité d’un événement

Dans le cas équiprobable : \[ P(A) = \frac{\text{nombre d'issues favorables}}{\text{nombre total d'issues}} \]
\[ 0 \leq P(A) \leq 1 \] \[ P(\Omega) = 1 \] \[ P(\varnothing) = 0 \]

4. Opérations sur les événements

Complémentaire : \[ P(\overline{A}) = 1 - P(A) \]
Formule de l’union : \[ P(A \cup B) = P(A) + P(B) - P(A \cap B) \]

5. Probabilité conditionnelle

Si \( P(B) \neq 0 \) : \[ P_B(A) = \frac{P(A \cap B)}{P(B)} \]
Formule des probabilités composées : \[ P(A \cap B) = P(B) \times P_B(A) \]

6. Indépendance

Deux événements \( A \) et \( B \) sont indépendants si : \[ P(A \cap B) = P(A)P(B) \]

7. Variables aléatoires discrètes

Une variable aléatoire \( X \) est une fonction qui associe un réel à chaque issue de \( \Omega \).

Espérance

Si \( X \) prend les valeurs \( x_1, \dots, x_n \) : \[ E(X) = \sum_{i=1}^{n} x_i P(X = x_i) \]
Linéarité : \[ E(aX + b) = aE(X) + b \]

Variance

\[ V(X) = E(X^2) - (E(X))^2 \]
\[ V(aX + b) = a^2 V(X) \]

8. Loi binomiale (programme de Terminale)

Si une expérience comporte deux issues (succès/échec) de probabilité \( p \), répétée \( n \) fois de manière indépendante, alors la variable aléatoire \( X \) suit une loi binomiale notée : \[ X \sim \mathcal{B}(n,p) \]
\[ P(X = k) = \binom{n}{k} p^k (1-p)^{n-k} \]
\[ E(X) = np \] \[ V(X) = np(1-p) \]
